Monkey_Problem

/*English*/
This Script allows the user to test the program. It can be used introducing an input parameter to change the number of instances of the class "Monkey" that will run once the program has started (By default the number of threads is 8).
Example to create only 3 instances:
macOS: ./Monkey_Problem.sh 3
Linux: sh Monkey_Problem.sh 3
Windows with WSL: bash Monkey_Problem.sh 3

/*Spanish*/
Este script permite un parámetro de entrada para seleccionar el número 
de instancias de la clase "Monkey" que se van a crear.
Si no se inserta ningún parámetro por defecto se crearan 8 instancias.
Ejemplo para 3 instancias de "Monkey"
macOS: ./Monkey_Problem.sh 3
Linux: sh Monkey_Problem.sh 3
Windows with WSL: bash Monkey_Problem.sh 3
